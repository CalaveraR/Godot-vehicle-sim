# Godot-vehicle-sim
Simulation of vários partes of motorized vehicles tô work below or above the basic system on Godot, something like a physics wrapper, giving it a alternative way of behavior using a chained reaction based on rigidbodies to emulate some behavior of softbodies


## Repository conventions
- Canonical folder for engine scripts: `engine/` (the duplicate `Engine/` tree was removed to avoid drift).
